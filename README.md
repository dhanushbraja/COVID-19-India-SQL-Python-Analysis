# COVID-19 India SQL & Python Analysis

## Project Overview

This project analyzes COVID-19 India data using MySQL and Python. The dataset is stored in a MySQL database, queried using SQL, and visualized with Python libraries such as Pandas and Matplotlib.

---

## Objectives

- Import COVID-19 dataset into MySQL.
- Connect Python with MySQL.
- Execute SQL queries using Python.
- Analyze confirmed, cured, and death cases.
- Create visualizations for better insights.

---

## Technologies Used

- MySQL Workbench
- SQL
- Python
- Pandas
- Matplotlib
- Jupyter Notebook

---

## Dataset

The dataset contains COVID-19 records across Indian states and union territories, including:

- Date
- State/Union Territory
- Confirmed Cases
- Cured Cases
- Death Cases

---

## SQL Analysis

The project includes SQL queries for:

- Total Confirmed Cases
- Total Cured Cases
- Total Death Cases
- State-wise Confirmed Cases
- State-wise Cured Cases
- State-wise Death Cases

---

## Python Visualizations

- State-wise Confirmed Cases (Bar Chart)
- State-wise Death Cases (Horizontal Bar Chart)
- State-wise Cured Cases (Line Chart)
- Top States by Confirmed Cases (Pie Chart)
- Confirmed vs Deaths (Scatter Plot)
